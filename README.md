# Name）
 ~ 曲選曲 ~
 
曲を選択し、閲覧できる掲示板
 
# DEMO
 
曲やジャケットのいいね、閲覧ができる。いろんな曲を知るきっかけになる。
 
# Features
 
Login機能、アカウント編集機能、曲の並び替え、いいね機能など
